MyCalcus Project

This is a simple JavaFX calculator application built using the MVC (Model-View-Controller) architecture.

Features
- Basic arithmetic operations (+, -, x, ÷)
- Square root function
- Clear and backspace functionality
- Real-time expression display

Structure
- View: FXML file (calcus-view.fxml) handles the UI layout
- Controller: CalcusController.java handles user interactions and events
- Model: Math logic is implemented within the controller

How to Run
Open the project in IntelliJ IDEA and run the CalcusApplication class.
